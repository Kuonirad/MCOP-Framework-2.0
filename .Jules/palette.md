## 2025-12-19 - Mirroring Hover Effects for Keyboard Focus
**Learning:** Directional cues like arrows (→) that animate on hover provide a sense of progression. Keyboard users miss this feedback if the animation is restricted to hover. Adding `group-focus-visible:translate-x-1` mirrors this experience, making the interface feel more responsive and consistent for all users.
**Action:** Always check if hover animations, especially directional ones, should have a corresponding focus state.
