# Polymathic Palette Journal

## 2025-05-21 - Visual Feedback for Keyboard Users

**Learning:** Hover effects like moving arrows (→) provide excellent directional cues, but keyboard users often miss out on this feedback. Adding `group-focus-visible:translate-x-1` ensures users navigating via Tab key get the same delightful confirmation as mouse users.

**Action:** Always mirror hover transforms with `focus-visible` states for interactive elements, especially directional indicators.
