# Palette's Journal

## 2026-02-26 - Focus Visible Directional Cues
**Learning:** Hover effects that indicate direction (like moving arrows) are often missed by keyboard users, creating an unequal experience.
**Action:** Always mirror `group-hover:translate-x-*` with `group-focus-visible:translate-x-*` on interactive elements to ensure keyboard users receive the same directional affordance.
