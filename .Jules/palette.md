# Palette's Journal

## 2025-05-15 - Consistent Directional Feedback
**Learning:** Keyboard users often miss out on subtle directional cues that are only available on hover. When a link implies movement (like an arrow), animating it on focus as well as hover creates a more consistent and polished experience for all users.
**Action:** Always pair `group-hover:translate-x-1` with `group-focus-visible:translate-x-1` for directional icons inside interactive elements.
