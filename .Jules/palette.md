# Palette Journal

## 2025-02-18 - Directional Cues and Focus States
**Learning:** Decorative directional cues (like arrows) that animate on hover often lack equivalent feedback for keyboard users, making the interface feel static and less responsive to non-mouse interactions.
**Action:** When adding `group-hover` translations to icons, always pair them with `group-focus-visible` to ensure keyboard users receive the same delightful "nudging" feedback.
