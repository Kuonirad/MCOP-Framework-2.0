# Palette's Journal

## 2025-10-26 - Keyboard Parity for Hover Animations
**Learning:** Interactive elements often have delightful hover animations (like translating arrows) but fail to provide the same feedback to keyboard users, creating a second-class experience.
**Action:** Always pair `group-hover:translate-x-1` with `group-focus-visible:translate-x-1` to ensure parity.
