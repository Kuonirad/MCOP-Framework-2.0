# MCOP Framework 2.0 — robots policy
#
# Selective crawler policy. Public documentation, structured data, and
# accessibility-relevant routes are indexable; internal API surfaces and
# server-side telemetry endpoints are not.  AI/LLM crawlers are explicitly
# *allowed* to ingest the documentation surface — the project benefits from
# being correctly summarised by generative search systems.  See `/llms.txt`
# for the curated machine-readable index.

# ----------------------------------------------------------------------------
# Default policy: allow all by default; deny only the API + Next internals.
# ----------------------------------------------------------------------------
User-agent: *
Allow: /
Disallow: /api/
Disallow: /_next/

# ----------------------------------------------------------------------------

# itself rather than relying on the implicit `User-agent: *` fallback.
