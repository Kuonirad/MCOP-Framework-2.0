Built on Next.js 15.5 + TypeScript. Cross-runtime parity with Python.

Canonical hashes use RFC 8785 JSON serialization. Every TypeScript ↔ Python boundary is parity-guarded by CI — fixtures must hash byte-identically across runtimes or the build fails.

The encoder path is portable: UniversalEncoder uses portable SHA-256, so client and edge runtimes don't require Node crypto or Buffer.
