# v2.0.1 — Onboarding & Dependency Maintenance

**Tag target:** `d2c7a14` (2025-12-31)

First patch release after the v2.0 launch. Focuses on making the project
welcoming to new contributors and modernizing the CI dependency surface.

## Added

- `CONTRIBUTOR_ONBOARDING.md` — a 30-minute runway covering setup, local
  development, and a first-PR walkthrough.
- `ROADMAP_TO_100.md` — historical aspirational roadmap (retained for
  context; see header note).
- Automated eco-fitness scoring system (`npm run eco:audit`) plus a
  self-referential meta-trace integration.
- `package-lock.json` generated from the full dependency audit.

## Changed

- CI actions bumped to v6: `actions/checkout`, `actions/setup-node`,
  `github/codeql-action`, `actions/upload-artifact`.
- Framework: `next` bumped `16.0.10 → 16.1.0`.
- Development-group dependencies updated (`@types/node` and co.).

## Fixed

- Docs-button accessibility: removed the fixed width, added a hover/animation
  affordance, and pinned the directional arrow.
- Dark-mode icon visibility and responsive sizing across the landing page.
